# IoBroker.fullcalendar
带有 [全日历](https://fullcalendar.io) 的时间表。

您不得使用任何外部资源，调度仅在 ioBroker 中处理，不会与“谷歌日历”或“iTunes”等任何外部服务结合。

![例子](../../../en/adapterref/iobroker.fullcalendar/img/example.png)

您可以使用日历控制您的事件，并可以计划定期控制它们。
